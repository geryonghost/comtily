# Configure Dev Environment
```
npm install express
npm install ejs
npm install axios
npm install vhost
npm install nodemon
```

# Local Hosts
## Mac
`sudo nano /private/etc/hosts`

## Windows
`runas /user:<localmachinename>\administrator notepad c:\windows\systems\drivers\etc\hosts`
